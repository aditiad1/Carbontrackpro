Carbon Footprint Calculator - Embedding Options

Basic Embedding

This is the simplest way to embed the carbon footprint calculator on your website with default settings.

<iframe src="https://replit.com/@ameetsinhvaghel/CarbonTrackPro?embed=true" width="100%" height="1000" style="border: 1px solid #ddd; border-radius: 8px;" frameborder="0" scrolling="auto" title="Carbon Footprint Calculator" ></iframe>

Customized Embedding

Customize the appearance of the calculator to match your website's design using URL parameters.

Available URL Parameters

Parameter Options Description
embed true Enables embedded mode
theme light, dark Sets the color theme
showBranding true, false Shows or hides the calculator branding
parentSite your-domain.com Identifies your site for analytics
<iframe src="https://replit.com/@ameetsinhvaghel/CarbonTrackPro?embed=true&theme=light&showBranding=true&parentSite=your-domain.com" width="100%" height="1000" style="border: 1px solid #ddd; border-radius: 8px; box-shadow: 0 4px 8px rgba(0,0,0,0.1);" allow="camera; microphone; autoplay; encrypted-media; fullscreen; geolocation" frameborder="0" scrolling="auto" title="Carbon Footprint Calculator" ></iframe>

Troubleshooting

  • If the calculator doesn't load: Make sure to replace "YOUR_APP_URL" with your actual Replit app URL.
  • If you experience CORS issues: Your Streamlit app already has CORS enabled in the .streamlit/config.toml file.
  • For security errors: Some websites block iframes. You may need to modify your Content Security Policy to allow the embed.
  • For mobile responsiveness: Use the responsive embedding code and test on different devices.
